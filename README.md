# Calculator Project

A modern, responsive calculator built using **HTML, CSS, and JavaScript**, featuring **dark/light themes** and **keyboard support**.

## 🌟 Features
- Basic arithmetic: `+`, `-`, `*`, `/`, `%`
- Dark and Light theme toggle
- Keyboard support:
  - Numbers `0-9`
  - Operators `+ - * /`
  - `Enter` → `=`
  - `Backspace` → delete last input
  - `C` → clear
- Responsive grid layout for buttons
- Hover effects and modern UI
